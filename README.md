## 会場音響配線図

![配線図](docs/images/ag06_wiring-Page-1.svg)

### パターンA（ワイヤレスもZoomへ送る場合）
ATW-SP707aの音声出力にAUXの音が混ざらないことを現地確認した上で使用。

### パターンB（ワイヤレスは会場専用・推奨）
シンプルで確実。ループリスクなし。

### 必要なケーブル
- XLRケーブル ×2（マイク→AG06）
- USBケーブル（AG06付属）
- 6.3mm TRS → RCA赤白 変換ケーブル（AG06 MONITOR OUT → ATW AUX入力）
- 6.3mm TS → TRS 変換（パターンAのみ・ATW音声出力 → AG06 CH3）
